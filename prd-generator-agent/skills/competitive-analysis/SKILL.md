# Competitive Analysis

## Purpose
Identify the competitive landscape and find AI-driven differentiation opportunities.

## Instructions

Using research findings and the target company's domain:

### 1. Competitor Identification
Identify 3–5 direct competitors and 2–3 indirect competitors:
- Company name and URL
- Market position (leader, challenger, niche)
- Key differentiators
- Target market overlap

### 2. Feature Comparison
Create a comparison matrix:
- Core features across competitors
- AI/ML capabilities each competitor offers
- Pricing positioning
- Integration ecosystem

### 3. AI Gap Analysis
Identify AI opportunities competitors are missing:
- Automation gaps (manual processes competitors haven't automated)
- Intelligence gaps (decisions made without data/AI support)
- Experience gaps (user experiences that could be personalized)
- Operational gaps (internal processes ripe for AI)

### 4. Differentiation Strategy
Recommend 3–5 AI-powered differentiators:
- Feature description
- Why competitors haven't done this
- Technical feasibility (low / medium / high)
- Business impact (low / medium / high)
- Implementation priority

## Output Format
Structured markdown with comparison tables and prioritized recommendations.
